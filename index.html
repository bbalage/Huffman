Huffman kódolás

A kódolás elkészítéséhez válassza ki az input fájlt!
Az input fájl JSON formátumú és a lentebb elérhető példafájlok felépítését követik. Szükséges a kód ABC elemszámának megadása, továbbá egy lista a forrás ABC elemeivel és azok valószínűségével. A "Feldolgozás eredménye" felirat alatt a feltöltés után meg fog jelenni az eloszlás, a generált kódok, a használt kód ABC elemek, továbbá a számított entrópia, átlagos kódhossz és hatásfok. Amennyiben ezeket az adatokat egy JSON fájlban le kívánja tölteni, a "Letöltés" gombra kattintva ezt megteheti.
A forráskód elérhető a https://github.com/bbalage/Huffman linken.
